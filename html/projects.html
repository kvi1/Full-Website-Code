2) Password Generator

Developed in Python, and using key elements such as user input, strings, ASCII, conditionals, and others, this is a random password generator. Based on the type of password the user wants, the program outputs a password with or without a custom phrase if the user wants one, and lets the user know if the password is weak, medium, or strong.
